Inline Media Queries

Write inline styles for specific media queries. Currently supported are the same media queries as used in Bootstrap v. 3.x:

  • xs (less than 768px)
  • sm (equal and greater than 768px)
  • md (equal and greather than 992px)
  • lg (equal and greater than 1200px)

Basic usage

Simple add data attributes to the desired HTML element with css styles

<h3 	
 data-style-lg="background-color: black; color: white;" 
 data-style-md="background-color: green; color: white;"
 data-style-sm="background-color: yellow; color: black;" 
 data-style-xs="background-color: blue; color: white;"
>
 Resize your browser window to see how it's working
</h2>

Exclude elements from inline media queries

In case you might come across display problems or just need to hide the styles applied by inline media queries you can turn effects off by adding the following data attribute to your element: 

<h3 
 data-style-exclude
 data-style-lg="background-color: black; color: white;" 
 data-style-md="background-color: green; color: white;"
 data-style-sm="background-color: yellow; color: black;" 
 data-style-xs="background-color: blue; color: white;"
>
